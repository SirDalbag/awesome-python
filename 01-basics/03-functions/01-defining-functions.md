# Определение функций

Функции в Python определяются с использованием ключевого слова `def`, за которым следует имя функции и круглые скобки с возможными параметрами.

## Пример простой функции

```python
def greet():
    print("Hello, World!")
```

## Вызов функции

```python
greet()  # Ожидаемый вывод: Hello, World!
```
